ikeaOS

A minimal Debian remix featuring KDE Plasma and Firefox.

About ikeaOS

ikeaOS is a lightweight, minimal operating system built on Debian. It includes only the core KDE Plasma desktop environment and the Firefox web browser, providing a clean base for further user customization.
